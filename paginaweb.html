Introdução ao HTML

Indice

O HTML (HyperText Markup Language) é a linguagem padrão para criar páginas web.
Ele utiliza uma estrutura baseada em tags para organizar e exibir conteúdo em navegadores.
Entender seus fundamentos é essencial para quem deseja trabalhar com desenvolvimento web.

1. Estrutura Básica de um Documento HTML

Indice

Todo documento HTML segue uma estrutura básica. Veja o exemplo abaixo: 

                        
                <!DOCTYPE html>
                <html>
                <head>
                    <title>Minha Primeira Página</title>
                </head>
                <body>
                    <h1>Bem-vindo ao HTML</h1>
                    <p>Esta é uma página simples criada com HTML.</p>
                </body>
                </html>
                        
                    
            
            <!DOCTYPE html>: Declaração do tipo de documento, informando que é um HTML5.
            <html>: Tag que envolve todo o conteúdo da página.
            <head>: Contém informações meta sobre a página, como título e links para estilos.
            <body>: Abriga o conteúdo visível da página.

Principais Tags do HTML

Indice

Cabeçalhos Utilizados para títulos e subtítulos, variando de <h1> (mais importante) a <h6> (menos importante).

<h1>Título Principal</h1>
<h2>Subtítulo</h2>

Parágrafos

Usados para texto comum: <p>Este é um parágrafo.</p>

Links

Permitem a navegação entre páginas ou recursos externos: <a href="https://example.com">Clique aqui</a>

Imagens Exibem imagens na página: <img src="imagem.jpg" alt="Descrição da imagem">

Listas

Ordenadas (<ol>):
<ol> <li>Primeiro item</li>
<li>Segundo item</li> </ol>

Não ordenadas (<ul>):
<ul> <li>Item 1</li>
<li>Item 2</li> </ul>

3. Atributos

Indice

As tags HTML podem ter atributos para fornecer informações adicionais. Exemplos comuns incluem:

  • href: Especifica o destino de um link.
  • src: Define a origem de uma imagem.
  • alt: Fornece texto alternativo para imagens.
  • class e id: Usados para estilização e identificação de elementos.

Exemplo

<a href="https://example.com" target="_blank">Visitar site</a>

4. Boas Práticas

Indice

Sempre feche as tags, mesmo as opcionais (ex.: <html>).
Use indentação para facilitar a leitura do código.
Adicione texto alternativo às imagens para acessibilidade.

Conclusão

Indice

Com esses conceitos básicos, você já pode começar a criar páginas simples em HTML.
É importante praticar para entender como as tags interagem e experimentar novos elementos à medida que avança.
